Frequently Asked Questions

Everything you need to know about RepoDent

RepoDent is a comprehensive student management system designed for educational institutions. It provides separate dashboards for teachers, students, and parents with features like attendance tracking, grade management, and communication tools.

You can access the demo by using the provided credentials: Teacher (teacher@sms.edu / teacher123), Student (student@sms.edu / student123), or Parent (parent@sms.edu / parent123). Click on any role card or use the login dropdown in the navigation.

RepoDent includes smart attendance tracking, grade management, timetable system, analytics dashboard, communication tools, report generation, and multi-role access for teachers, students, and parents.

Yes, RepoDent is an open source educational project built with modern web technologies. The source code is available on GitHub for learning and educational purposes.

The demo includes real sample data from Arya College of Engineering & IT, featuring students from III SEM CS-A, CS-B, and CS-C sections, faculty information, subject details, and realistic academic records for demonstration purposes.
